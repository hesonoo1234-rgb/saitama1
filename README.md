## 内容

・要約表示
### 説明
公約全てを約5分で比較できます。
何も考えずにテキトーに投票する人、どこに投票すればいいのかよくわからないで結局投票しない人が少しでも減れば嬉しいです。

今の政治家の票を集め方には問題があり、公約を中心として政治家に票が集まるシステムをつくれば日本はよりよい政治ができるようになると思い、このサイトを制作しました。

今後のこのサイト予定は、以下の通りです
1、このサイトのアプリ化
2、公約それぞれにそのメリットとデメリットを付け加える
3、議員ごとに詳細参照の要約指標*を作ること

*現時点で考えている項目は公約の明確さ、公約の実行状況、国会での活動記録、発言と公約の整合性、情報の裏付け度
3についてはどのようにデータを取るかなど、あまり深く考えていないため実装しない場合も十分あり得ます。(お金がたくさん集まれば実装できると思います。)
