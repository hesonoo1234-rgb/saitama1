# kouyaku
このサイトは日本の議員の公約をまとめたものです。（現在制作中）
## URL
https://hesonoo1234-rgb.github.io/kouyaku/
## 内容
国会議員の候補者の公約を要約し、選挙区ごとにまとめました。
### 説明
公約全てを約5分で比較できます。
何も考えずにテキトーに投票する人、どこに投票すればいいのかわからないず、結局投票しない人が少しでも減れば嬉しいです。

今の政治家の票を集め方には問題があり、公約を中心として政治家に票が集まるシステムをつくれば政治がより良くなると思い、このサイトを制作しました。
#### サイトの今後の予定
  今後のこのサイト予定は、以下の通りです  
1、日本の衆議院と参議院の公約の要約を完成させる（全都道府県）  
2、このサイトのアプリ化  
3、公約それぞれにそのメリットとデメリットを付け加える  
4、議員ごとに詳細参照の要約指標*を作る  

*現時点で考えている項目は公約の明確さ、公約の実行状況、国会での活動記録、発言と公約の整合性、公約の情報の裏付け度  
4についてはどのようにデータを取るかなど、あまり深く考えていないため実装しない可能性もあります。(今後、このサイトでお金をたくさん集めることができれば実装できると思います）
